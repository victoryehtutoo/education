
# Education Repository

Welcome to the **Education Repository** for Alin Tagar English School EDU.  
This repository contains resources, scripts, and documentation to support our educational programs, including:

- General English courses (ATGE Starter, Elementary, etc.)
- Academic English and Business English materials
- Tools for managing student reports and school operations

## Features
- PHP logic for student report generation
- CSS fixes for printing reports
- Free graded reader resources for Level 1 students
